# Напоминалка
Навайбкожено с помощью ИИ!
Android-приложение для напоминаний, которые «звонят» и озвучивают текст с помощью TTS (синтеза речи), вместо обычных push-уведомлений.

## Возможности

- Создание напоминаний с текстом и временем срабатывания
- При срабатывании: полноэкранный экран «звонка», вибрация, мелодия и озвучивание текста (TTS)
- Поддержка Bluetooth-гарнитуры: напоминание приходит как входящий звонок, можно ответить кнопкой на гарнитуре
- Настройки: выбор движка TTS, задержка перед озвучиванием, скорость речи, тестовое воспроизведение
- Массовое удаление напоминаний (режим выбора, «Выбрать все»)
- Автоудаление прошедших напоминаний (опция в настройках)
- синхронизация с календарями Android^ читает и записывает напоминания в календарь. Полезно при добавлении напоминаний через Ассистенты типа Gemini
- отложенные напоминания при отклонении звонка

## Как подружить напоминалку с Алисой

1. Создаём пароль приложения. Заходим [сюда](https://id.yandex.ru/), внизу страницы нажимаем ещё, потом находим заголовок безопасность. Далее ищем пароли приложений. Переходим.
2. Создаём пароль приложения для календаря. Сохраняем его.
3. Создайте какой-нибудь календарь в Яндекс календаре любым способом. Я создавал через одноимённое приложение  на Android.
4. Установите или откройте приложение умный дом от Яндекса, перейдите в настройки умного дома, продуктивность, будильники и напоминания, календари и выберите ваш свежесозданный календарь как основной 
5. Установите [это](https://www.davx5.com/download) приложение и настройте его следующим образом:
Добавить аккаунт, войти по url и имени пользователя,
url: https://yandex.ru
логин: ваш пароль из пароля приложений, который мы сделали на первом шаге
Теперь зайдите в свежесозданный аккаунт и настройте интервал 15 минут для календарей, чтобы они почаще синхронизировались.
На Яндекс станции или везде, где есть Алиса, просто скажите добавь в календарь, дождитесь ответа Алисы и следуйте дальнейшим инструкциям.

## Требования

- Android 7.0 (API 24) и выше
- Разрешения: уведомления, точные будильники (при необходимости — звонки для гарнитуры), доступ к календарям

## Сборка

```bash
./gradlew assembleDebug
```

Или откройте проект в Android Studio и соберите через меню Build.

## Технологии

- Kotlin, Jetpack Compose (основной экран), XML (экран звонка)
- AlarmManager, BroadcastReceiver, Telecom API (для гарнитуры), TextToSpeech
